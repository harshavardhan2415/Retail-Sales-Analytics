# Retail Sales Analytics

## Indian E-Commerce Customer Behavior Analysis

An end-to-end data analytics project that analyzes customer behavior, sales performance, and operational metrics for an Indian e-commerce business. The project demonstrates the complete analytics lifecycle—from data cleaning and SQL analysis to interactive Power BI dashboards and business recommendations.

---

## Business Problem

An Indian e-commerce company has experienced slower revenue growth despite increasing website traffic. The objective of this project is to analyze customer behavior, purchasing patterns, product performance, and operational metrics to identify actionable insights that support better business decisions.

---

## Project Objectives

* Increase revenue
* Improve customer engagement
* Reduce cart abandonment
* Improve customer satisfaction
* Identify top-performing product categories
* Optimize payment methods
* Understand regional sales performance
